Welcome to the About page


Scansion is a tool for the metrical analysis of poetry.

In order to use Scansion, simply type in or copy and paste a text into the text are on the main page and click 'Scan Text'.

Scansion will then scan your text for any recurring metrical patterns, and tell you if it thinks it's found any.

From there, you can see its analysis in more detail, highlight the stress patterns in bold or by colour, or edit the text by swapping individual words. Alternatively, pressing 'Scansion' in the header will return you to the main entry screen.

Scansion was built by Stuart Rawlinson as part of an MSc in Software Engineering. It is powered by the PROSODIC Python library and the Wordnik API.
